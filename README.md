# 🎒 tollypips

**Your all-in-one backpack of tools. Fast, free, and unapologetically bold.**

tollypips is a collection of utilities designed specifically to turbocharge workflows for students and developers alike. Featuring a highly interactive, Neo-Brutalist design.

---

## ✨ Features

- **⚡ Fast & Free:** Access premium-level tools without spending a dime.
- **🔓 No Login Required:** Jump straight into the action. No tedious sign-ups or email verifications needed.
- **🎨 Unapologetically Bold:** A stunning Neo-Brutalist design language built with Framer Motion and Tailwind CSS v4.
- **🛠️ The Arsenal:** A growing collection of highly useful tools and browser extensions.

## 🧰 The Toolkit

1. **[Context Passport](https://github.com/iamshubham1704/passport.git)**
   - An intelligent browser extension to save, compress, and transfer your AI chat context across different platforms (ChatGPT, Claude, Gemini, etc).

2. **PDF Editor** _(Coming Soon)_
   - Quickly modify, merge, and split PDF documents right in your browser. No server uploads required.

3. **Attendance Manager** _(Coming Soon)_
   - Track your university attendance with ease. Get smart predictions on how many classes you can afford to miss.

## 🚀 Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Animations:** Framer Motion
- **Icons:** Lucide React

## 💻 Running Locally

1. **Clone the repository:**

   ```bash
   git clone https://github.com/yourusername/tollypips.git
   cd tollypips
   ```

2. **Install dependencies:**

   ```bash
   npm install
   ```

3. **Start the development server:**

   ```bash
   npm run dev
   ```

4. **Explore the app:** Open `http://localhost:3000` in your browser.
